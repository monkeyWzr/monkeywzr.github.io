## Basic

```racket
# lang racket
#lang racket
(provide (all-defined-out))

;this is a comment

(define s "hello")

(define x 3)
(define y (+ x 2))

(define cube1
  (lambda (x)
    (* x (* x x))))

(define cube2
  (lambda (x)
    (* x x x)))

(define (cube3 x)
  (* x x x))

(define (pow1 x y)
  (if (=y 0)
      1
      (* x (pow1 x (- y 1)))))

; currying
(define pow2
  (lambda (x)
    (lambda (y)
      (pow1 x y))))

```

### List

* Empty list: `null`
	* `()` doesn"t work for `null` but `'()` does
* build a list: `(list e1 ... en)`
* Constructor: `cons`
* Access head of list: `car`
* Access tail of list: `cdr`
* Check for empty: `null?`
