Kali下Metasploit学习笔记

2015-12-07
kali

启动

使用框架前先开启其几个服务：

# service postgresql start
# msfdb init

注意kali2.0开始不再有metasploit服务，所以官方文档说要用msfdb init代替service metasploit start

然后启动msf控制台

# msfconsole

一些命令

workspace -h 帮助 workspace [-a/d/r] 创建/删除/重命名工作平台
